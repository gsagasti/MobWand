# MobWand

Minecraft Bedrock addon that allows you to control hostile mobs with a magic wand.

## Usage

### Get the MobWand
```
/scriptevent mobwand:give
```

### Controls
- **Left-click hostile mob**: Add to list (max 10 types)
- **Shift + right-click**: Open saved mobs menu
- **Hold the wand**: Pushes saved mobs every 3 seconds

## Requirements

- Minecraft Bedrock 1.21.0+
- Beta APIs enabled
